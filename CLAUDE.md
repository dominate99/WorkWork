# WorkWork

This repository packages the `ww-subagent-orchestrator` workflow as a plugin-style skill set.

When a task starts with `$ww`, or when the user is asking for orchestration, planning, dispatch, or review coordination, start with:

`plugins/workwork/skills/ww-subagent-orchestrator/SKILL.md`

Use the README for installation and packaging context, and use the `docs/` tree for persisted plans, specs, and review artifacts.
