# Legacy Superpowers Artifacts

This directory stores historical workflow artifacts that were created before the active `$ww` and `$www` cutover to `docs/cases/...`.

Archived families:

- `working-briefs/`
- `dispatch-plans/`
- `specs/`
- `plans/`

Rules:

- treat these files as historical references
- do not write new round artifacts here
- active rounds belong under `docs/cases/<case-slug>/rounds/<round-slug>/`
- navigational path references inside archived files may be normalized to `docs/legacy/superpowers/...` when that improves archive browseability
- historical prose that explains the pre-archive source layout may still mention `docs/superpowers/...` when changing it would distort the original meaning
